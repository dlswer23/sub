##### Termianl 단축어 



+ ls -> 현재 디렉토리 확인
+ ls-l -> 옵션 +
+ ls-al -> 옵션 전체
+ ll -> ls-al
+ cd -> change Directory
+  / -> 경로 구분자
+ .. -> 부모 디렉토리(상위)
+ . -> 자식 디렉토리(자기 자신)
+ vi +-@ -> 수정  



예시 ) 내가 만약에 Downloads에 들어가고 싶다  그러면 cd. Download 를 입력하면 된다.
